# employee_attrition_project

## Employee Attrition Prediction
This project focuses on analyzing employee data and building machine learning models to predict employee attrition. The objective is to identify key factors driving employee turnover, enabling data-driven retention strategies.

---

## Overview
- Developed an end-to-end ML pipeline for attrition prediction
- Performed data cleaning, feature engineering, and exploratory analysis
- Trained and evaluated multiple classification models
- Extracted actionable insights affecting employee retention

---

## Key Features
- Data preprocessing and encoding of categorical variables
- Correlation analysis and feature selection
- Model training using Logistic Regression, Decision Tree, and Random Forest
- Performance evaluation using accuracy and confusion matrix
- Feature importance analysis to identify critical drivers

---

## Results
- Identified key drivers such as job satisfaction, income, and work environment
- Built predictive models capable of classifying attrition with strong performance
- Provided insights to help organizations reduce employee turnover
